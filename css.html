2. CSS Selectors

1- Selectors determine which HTML elements the styles will apply to. Common types include:

2- Universal Selector (*): Targets all elements.

*Type Selector: Targets specific elements

Class Selector (.class): Targets elements with a specific class

ID Selector (#id): Targets a unique element by its ID.

Group Selector (A, B): Combines multiple selectors.

Descendant Selector (A B): Targets elements inside another element.

4. CSS Box Model

The box model describes the structure of elements in terms of:

* Content: The actual content inside the box.

* Padding: Space between content and border.

* Border: The boundary surrounding the padding.

* Margin: Space outside the border between elements.

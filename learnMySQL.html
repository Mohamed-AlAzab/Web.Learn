Introduction to MySQL

What is MySQL?

MySQL is an open-source relational database management system (RDBMS) that uses Structured Query Language (SQL) to manage and manipulate data. It is widely used for web applications and data storage.

Database Concepts

Core Database Ideas

  • Table: A collection of rows and columns.
  • Row: A single record in a table.
  • Column: A field in a table representing a data attribute.
  • Primary Key: A unique identifier for a row.
  • Foreign Key: A field that links one table to another.

Basic MySQL Commands

Essential Commands

  • CREATE DATABASE: Create a new database.
  • USE: Select a database to work with.
  • CREATE TABLE: Create a table with defined columns.
  • INSERT INTO: Add records to a table.
  • SELECT: Retrieve data from a table.
CREATE DATABASE my_database;
USE my_database;
CREATE TABLE users (
  id INT AUTO_INCREMENT PRIMARY KEY,
  name VARCHAR(100),
  email VARCHAR(100)
);
INSERT INTO users (name, email) VALUES ('John Doe', 'john@example.com');
SELECT * FROM users;

MySQL Data Types

Types of Data

  • INT: Integer numbers.
  • VARCHAR: Variable-length strings.
  • DATE: Date values.
  • FLOAT: Floating-point numbers.

Writing Queries

Retrieving Data 

SELECT * FROM users WHERE name = 'John';
SELECT name, email FROM users ORDER BY name ASC;
SELECT COUNT(*) FROM users;

Joins in MySQL

Combining Tables 

SELECT orders.id, users.name, orders.total 
FROM orders 
JOIN users ON orders.user_id = users.id;

Indexes

Indexes speed up queries on large tables. Create with:

CREATE INDEX idx_name ON users (name);

Transactions

Ensure data integrity with transactions:

START TRANSACTION;
UPDATE accounts SET balance = balance - 100 WHERE id = 1;
UPDATE accounts SET balance = balance + 100 WHERE id = 2;
COMMIT;

Practice Examples

Try these queries:

-- Retrieve all emails
SELECT email FROM users;

-- Get total number of users
SELECT COUNT(*) FROM users;

-- List all orders for a specific user
SELECT * FROM orders WHERE user_id = 1;

Conclusion

MySQL is a powerful database system. With practice, you'll be able to manage, query, and optimize databases efficiently!
